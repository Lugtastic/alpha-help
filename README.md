### **≡ MUTE ≡** 

##### ◇***INFORMATION:*** 
###### This command has [3] total arguments, one of which is optional.

##### ◇***COMMAND:*** 
###### $mute [user; @ or ID] [time; 1s, 1m, 1d, 1w etc.] [reason; optional no length min./max.]

##### ◇***EXAMPLE(S):*** 

###### $mute @User#0000 4d 

###### $mute 825213635201990710 30m Spammed in chat

###### $unmute [user; @ or ID]
